# Learning Content Management System

This is my Java OOP lab assignment project. In this project I made a Learning Content Management System. This code is about different types of learning content like VideoLesson, CodingExercise and FlashCardSet. I used one abstract parent class which is LearningContent and other three are child classes. Total there are 5 files in this project: LearningContent.java, VideoLesson.java, CodingExercise.java, FlashCardSet.java and Main.java. 

In this code I used OOP concepts like abstraction, inheritance, method overriding, constructor chaining, final method, this keyword, encapsulation, upcasting, downcasting, array of parent references, instanceof and toString method. The program first creates objects, then stores them in array, shows data before modification, updates values with loops, and then shows data after modification.
